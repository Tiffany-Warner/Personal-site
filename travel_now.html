Travel Now Mobile App Design

This project was completed as part a group project for CS 352-Usability Engineering. The goal was to create a user-interface based on usability design principles- namely Jakob Nielsen's Heuristic Guidelines.

Travel Now is a fictional mobile application that was designed be an all-in-one app for people to be able to plan an entire trip using only one application. The software used to help generate the final prototype was JustInMind.

My role in creating this prototype could be best described as lead designer. We all pitched ideas and the group decided on my design describing it as "easy to navigate and minimalistic." We performed analytical analysis on the prototype and conducted interviews for our empirical studies where we watched a user interact with our prototype. We then implemented changes in the design based on feedback we recieved and usability issues that become apparent through the analytical analyses. I created the final prototype with JustInMind and also the storyboard implementing all the design changes. The final report was created by all group members.
